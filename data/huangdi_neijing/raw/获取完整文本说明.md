# 《黄帝内经》完整文本获取说明

## 快速获取方法

### 方法1: 使用提供的脚本（推荐）

运行以下命令：
```bash
python scripts/fetch_huangdi_neijing.py
```

### 方法2: 手动下载

#### 推荐网站：

1. **中国古籍网**
   - 网址: https://www.guoxue.com/
   - 搜索"黄帝内经"
   - 下载TXT或PDF格式

2. **GitHub开源项目**
   - 搜索: `黄帝内经 txt` 或 `huangdi neijing`
   - 推荐项目: https://github.com/garychowcmu/ChineseText
   - 直接下载 `素问.txt` 和 `灵枢.txt`

3. **国学资源网**
   - 网址: https://www.guoxue.com/
   - 提供完整的古籍文本

#### 文件要求：

1. **文件名**:
   - `suwen.txt` - 素问（应包含81篇）
   - `lingshu.txt` - 灵枢（应包含81篇）

2. **编码**: 必须是 UTF-8 编码

3. **格式**: 
   - 每篇以 `## 第X篇 篇名` 开头（推荐）
   - 或 `第一篇 篇名` 开头
   - 正文内容紧跟其后

4. **保存位置**: 
   ```
   data/huangdi_neijing/raw/suwen.txt
   data/huangdi_neijing/raw/lingshu.txt
   ```

## 验证文件完整性

运行以下命令检查：
```bash
python -c "import os; files = ['suwen.txt', 'lingshu.txt']; dir = 'data/huangdi_neijing/raw'; [print(f'{f}: {os.path.getsize(os.path.join(dir, f))} bytes') if os.path.exists(os.path.join(dir, f)) else print(f'{f}: 不存在') for f in files]"
```

**预期大小**:
- 素问: 通常 > 100KB
- 灵枢: 通常 > 100KB

## 初始化知识库

获取完整文本后，运行初始化脚本：
```bash
python core/tools/init_huangdi_kb.py
```

这将：
1. 解析文本文件
2. 生成结构化数据
3. 构建向量数据库

## 常见问题

### Q: 文件下载后显示乱码？
A: 确保文件是UTF-8编码。可以用文本编辑器（如VS Code、Notepad++）打开，选择"编码" -> "UTF-8"保存。

### Q: 文件格式不对？
A: 确保每篇以明确的标题开头，如 `## 第一篇 上古天真论`。可以使用文本处理工具调整格式。

### Q: 从哪里可以快速获取？
A: 推荐从GitHub搜索 `黄帝内经 txt`，通常能找到开源的完整文本文件。

## 提示

如果暂时无法获取完整文本，系统会使用示例文件（`suwen_sample.txt` 和 `lingshu_sample.txt`），但功能会受限。建议尽快获取完整文本以获得最佳体验。
