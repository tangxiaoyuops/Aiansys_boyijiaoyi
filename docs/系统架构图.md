## 数据流图

### 回测数据流

```
用户触发回测
      │
      ▼
┌─────────────┐
│ 配置回测参数 │ (期货代码、策略、参数、时间范围)
└─────────────┘
      │
      ▼
┌─────────────┐
│ 获取历史数据 │ (从API/缓存获取OHLCV+持仓量数据)
└─────────────┘
      │
      ▼
┌─────────────┐
│ 初始化回测   │ (创建引擎、券商、组合、风险管理器)
└─────────────┘
      │
      ▼
┌──────────────────────────────────────┐
│      逐Bar回测循环                    │
│  ┌────────────────────────────────┐  │
│  │ 1. 更新持仓价值                 │  │
│  │ 2. 风险检查（保证金率、强平）   │  │
│  │ 3. 策略生成信号                 │  │
│  │ 4. 执行订单（滑点、手续费）     │  │
│  │ 5. 更新组合状态                 │  │
│  │ 6. 记录权益和交易                │  │
│  └────────────────────────────────┘  │
│         │                             │
│         ▼ (循环直到数据结束)          │
└──────────────────────────────────────┘
      │
      ▼
┌─────────────┐
│ 计算绩效指标 │ (30+项指标)
└─────────────┘
      │
      ▼
┌─────────────┐
│ 返回结果     │ {
│             │   equity_curve: [],
│             │   trade_log: [],
│             │   metrics: {},
│             │   ...
│             │ }
└─────────────┘
      │
      ▼
┌─────────────┐
│ 前端渲染     │ (权益曲线、回撤图、交易日志等)
└─────────────┘
```
