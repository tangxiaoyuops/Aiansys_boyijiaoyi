## 📊 关键指标速查表

### 收益类指标
| 指标 | 计算公式 | 意义 |
|-----|---------|------|
| 总收益率 | (最终权益 - 初始资本) / 初始资本 × 100% | 策略总体盈利能力 |
| 年化收益率 | 总收益率 × (365 / 交易天数) | 标准化年度收益 |
| 月度收益率 | 每月收益 / 月初资金 × 100% | 月度表现跟踪 |

### 风险类指标
| 指标 | 计算公式 | 意义 |
|-----|---------|------|
| 最大回撤 | Max(累计最高权益 - 当前权益) / 累计最高权益 | 最大损失幅度 |
| 波动率 | std(日收益率) × √252 | 收益波动程度 |
| VaR (95%) | 日收益率的5%分位数 | 单日最大损失（95%置信度） |
| CVaR (95%) | VaR以下收益的平均值 | 极端情况下的期望损失 |

### 风险调整收益
| 指标 | 计算公式 | 优秀标准 |
|-----|---------|---------|
| 夏普比率 | (年化收益 - 无风险利率) / 年化波动率 | > 2.0 |
| 索提诺比率 | (年化收益 - 目标收益) / 下行波动率 | > 3.0 |
| 卡玛比率 | 年化收益 / 最大回撤 | > 1.0 |

### 交易统计
| 指标 | 计算公式 | 优秀标准 |
|-----|---------|---------|
| 胜率 | 盈利交易次数 / 总交易次数 × 100% | > 50% |
| 盈亏比 | 平均盈利 / 平均亏损 | > 2.0 |
| 盈利因子 | 总盈利 / 总亏损 | > 1.5 |

## 💻 代码速查

### 1. 快速计算夏普比率

```python
def calculate_sharpe_ratio(equity_curve: List[float], 
                          risk_free_rate: float = 0.03,
                          periods_per_year: int = 252) -> float:
    """
    计算夏普比率
    
    Args:
        equity_curve: 权益曲线
        risk_free_rate: 无风险利率（年化）
        periods_per_year: 每年交易周期数（日线252，小时线252*4）
    
    Returns:
        夏普比率
    """
    returns = np.diff(equity_curve) / equity_curve[:-1]
    
    # 年化收益率
    annual_return = np.mean(returns) * periods_per_year
    
    # 年化波动率
    annual_volatility = np.std(returns) * np.sqrt(periods_per_year)
    
    # 夏普比率
    sharpe = (annual_return - risk_free_rate) / annual_volatility
    
    return sharpe
```

### 2. 快速计算最大回撤

```python
def calculate_max_drawdown(equity_curve: List[float]) -> Dict[str, float]:
    """
    计算最大回撤及相关信息
    
    Returns:
        {
            'max_drawdown': 最大回撤（百分比）,
            'max_dd_duration': 回撤持续天数,
            'current_drawdown': 当前回撤
        }
    """
    equity = np.array(equity_curve)
    
    # 累计最高权益
    cumulative_max = np.maximum.accumulate(equity)
    
    # 回撤序列
    drawdown = (equity - cumulative_max) / cumulative_max
    
    # 最大回撤
    max_dd = np.min(drawdown)
    max_dd_idx = np.argmin(drawdown)
    
    # 回撤开始点（之前的峰值）
    peak_idx = np.argmax(equity[:max_dd_idx+1])
    dd_duration = max_dd_idx - peak_idx
    
    return {
        'max_drawdown': float(max_dd * 100),
        'max_dd_duration': int(dd_duration),
        'current_drawdown': float(drawdown[-1] * 100)
    }
```

### 3. 简单双均线策略示例

```python
class DualMAStrategy(Strategy):
    """双均线策略"""
    
    def _on_initialize(self):
        self.fast_period = self.get_parameter('fast_period', 5)
        self.slow_period = self.get_parameter('slow_period', 20)
    
    def generate_signal(self, data: pd.DataFrame, portfolio) -> Optional[Dict]:
        if len(data) < self.slow_period:
            return None
        
        # 计算均线
        data['fast_ma'] = data['close'].rolling(self.fast_period).mean()
        data['slow_ma'] = data['close'].rolling(self.slow_period).mean()
        
        current = data.iloc[-1]
        previous = data.iloc[-2]
        
        # 金叉信号
        if (previous['fast_ma'] <= previous['slow_ma'] and 
            current['fast_ma'] > current['slow_ma']):
            return {
                'action': 'OPEN_LONG',
                'size': 0.3,
                'reason': f'金叉: MA{self.fast_period} > MA{self.slow_period}'
            }
        
        # 死叉信号
        if (previous['fast_ma'] >= previous['slow_ma'] and 
            current['fast_ma'] < current['slow_ma']):
            return {
                'action': 'CLOSE_LONG',
                'reason': f'死叉: MA{self.fast_period} < MA{self.slow_period}'
            }
        
        return None
```
