# 大宗商品供给与价格系统分析 - 实现总结

## 实现概述

基于《大宗商品供给与价格系统分析-实现方案.md》，已完整实现整个系统，包括数据采集、结构化分析、自动搜索、技术指标计算、策略生成、回测评估等完整功能。

## 已实现模块

### 1. 数据模型（core/models/）

#### commodity_models.py
- **SupplyNode**: 供给节点（地区、产能、库存、贸易流等）
- **PricePoint**: 价格点（品种、合约、价格类型、价格值等）
- **NewsItem**: 新闻项（标题、摘要、来源、时间、标签等）
- **RawEvidence**: 原始证据包（供给链、价格、新闻列表）

#### strategy_models.py
- **StrategySignal**: 策略信号（类型、方向、入场、止损、仓位、置信度等）
- **StrategyPortfolio**: 策略组合（组合ID、策略列表、风险指标等）
- **BacktestResult**: 回测结果（收益率、夏普比率、最大回撤、净值曲线等）
- **TechnicalIndicators**: 技术指标（MA、MACD、RSI、布林带、ATR等）
- **RiskMetrics**: 风险指标（VaR、最大回撤、夏普比率等）

#### commodity_state.py
- **CommodityAnalysisState**: 完整的状态定义，包含所有工作流所需字段

### 2. 工具模块（core/tools/）

#### commodity_fetcher.py
- **CommodityFetcher**: 数据采集器
  - 采集供给链数据（产能、库存、贸易流）
  - 采集价格数据（现货、期货、指数）
  - 采集新闻数据（政策、天气、地缘、供需）
  - 支持模拟数据（可替换为真实数据源）

#### commodity_search.py
- **CommoditySearcher**: 搜索器
  - 执行自动搜索查询
  - 支持多搜索引擎切换
  - 返回搜索结果（标题、摘要、URL）

#### technical_indicators.py
- **TechnicalIndicatorCalculator**: 技术指标计算器
  - SMA/EMA（简单/指数移动平均）
  - MACD（指数平滑异同移动平均线）
  - RSI（相对强弱指标）
  - 布林带（Bollinger Bands）
  - ATR（平均真实波幅）
  - OBV（能量潮）
  - KDJ（随机指标）
  - CCI（顺势指标）
  - 综合指标计算接口

#### backtest_engine.py
- **BacktestEngine**: 回测引擎
  - 历史数据回测
  - 仓位管理（固定比例、风险控制）
  - 止损止盈执行
  - 回测指标计算（收益率、夏普比率、最大回撤、胜率等）
  - 风险指标计算（VaR、索提诺比率、卡玛比率等）

### 3. Agent节点（core/agents/）

#### commodity_intent_agent.py
- 意图解析节点
- 解析品种/产业链/时间/策略类型
- 初始化状态参数

#### commodity_fetch_agent.py
- 数据采集节点
- 调用数据采集器获取原始证据
- 错误处理与降级

#### commodity_analysis_agent.py
- 结构化分析节点
- 使用LLM对原始证据进行结构化分析
- 生成摘要、分类、驱动因子、置信度

#### commodity_judgment_agent.py
- 自动判定节点
- 识别缺失数据、矛盾点、关键驱动
- 生成建议搜索查询

#### commodity_search_agent.py
- 搜索执行节点
- 执行建议搜索查询
- 将结果合并到证据包

#### technical_indicators_agent.py
- 技术指标计算节点
- 计算各类技术指标
- 识别市场状态（趋势/震荡/转折）

#### strategy_generation_agent.py
- 策略生成节点
- 综合基本面与技术面
- 生成买卖策略信号（入场、止损、止盈、仓位）

#### strategy_risk_agent.py
- 风险评估节点
- 评估策略风险
- 计算风险指标（VaR、最大回撤、夏普比率）

#### strategy_backtest_agent.py
- 策略回测节点
- 对生成的策略进行历史回测
- 计算回测结果

#### commodity_report_agent.py
- 报告生成节点
- 生成最终分析报告
- 生成结构化输出（JSON）

### 4. 工作流图（core/graph/）

#### commodity_analysis_graph.py
- **create_commodity_analysis_graph()**: 创建工作流图
  - 节点连接：intent → fetch → structured_analysis → judgment → (search → structured_analysis) | technical_indicators → strategy_generation → strategy_risk → strategy_backtest → report → END
  - 条件边：根据搜索查询和轮次决定是否继续搜索
- **run_commodity_analysis()**: 运行分析
  - 初始化状态
  - 执行工作流
  - 返回结果

### 5. API路由（server/routers/）

#### commodity.py
- **POST /api/commodity/analyze**: 综合分析
  - 支持品种、时间范围、用户问题、策略类型
  - 返回报告、结构化数据、策略、回测结果
- **POST /api/commodity/strategy**: 策略生成
  - 生成买卖策略
  - 返回策略信号和技术指标
- **POST /api/commodity/backtest**: 策略回测
  - 执行策略回测
  - 返回回测结果和曲线数据
- **GET /api/commodity/strategies**: 策略列表
  - 获取历史策略记录
  - 支持筛选和分页
- **GET /api/commodity/health**: 健康检查

## 系统架构

```
用户请求 → 意图解析 → 数据采集 → 结构化分析 → 自动判定
                                                    ↓
                                              搜索查询 → 搜索执行 → 回到结构化分析
                                                    ↓
                                              技术指标计算 → 策略生成 → 风险评估 → 策略回测 → 报告生成 → 返回结果
```

## 主要特性

### 1. 多轮搜索闭环
- 自动判定识别信息缺口
- 生成建议搜索查询
- 执行搜索并合并结果
- 限轮次控制（默认2轮）

### 2. 完整技术指标体系
- 趋势指标：MA、EMA、MACD、ADX
- 动量指标：RSI、KDJ、CCI
- 波动率指标：ATR、布林带
- 成交量指标：OBV

### 3. 多种策略类型
- 趋势跟踪策略
- 套利策略（跨期、跨品种、跨市场）
- 套期保值策略
- 事件驱动策略

### 4. 完整回测体系
- 历史数据回测
- 仓位管理（固定比例、风险控制）
- 止损止盈执行
- 回测指标计算
- 风险指标计算

### 5. 风险管理
- VaR计算（95%、99%）
- 最大回撤监控
- 夏普比率、索提诺比率
- 卡玛比率
- 仓位规模计算

## 使用示例

### API调用示例

#### 1. 综合分析
```bash
curl -X POST http://localhost:8000/api/commodity/analyze \
  -H 'Content-Type: application/json' \
  -d '{
    "commodity_or_chain": "原油",
    "strategy_type": "trend",
    "enable_backtest": true,
    "max_rounds": 2
  }'
```

#### 2. 策略生成
```bash
curl -X POST http://localhost:8000/api/commodity/strategy \
  -H 'Content-Type: application/json' \
  -d '{
    "commodity": "铜",
    "strategy_type": "arbitrage"
  }'
```

#### 3. 策略回测
```bash
curl -X POST http://localhost:8000/api/commodity/backtest \
  -H 'Content-Type: application/json' \
  -d '{
    "strategy_id": "原油_long_2026-03-07T12:00:00",
    "start_date": "2025-01-01",
    "end_date": "2026-03-01",
    "initial_capital": 1000000,
    "commission": 0.0001
  }'
```

### Python调用示例

```python
from core.graph.commodity_analysis_graph import run_commodity_analysis

result = run_commodity_analysis(
    commodity_or_chain="原油",
    strategy_type="trend",
    enable_backtest=True,
    max_rounds=2
)

if result["success"]:
    data = result["data"]
    print(f"报告: {data['report']}")
    print(f"策略数量: {len(data['strategies'])}")
    print(f"回测结果: {len(data['backtest_results'])}")
```
