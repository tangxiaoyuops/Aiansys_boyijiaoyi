## 数据结构

六十四卦数据存储在 `core/data/hexagrams.json` 文件中，格式如下：

```json
{
  "序号": {
    "name": "卦名",           // 如：乾、坤、震等
    "full_name": "全名",      // 如：乾为天、水山蹇等
    "inner": "内卦名",        // 下卦：乾、坤、震、巽、坎、离、艮、兑
    "outer": "外卦名",        // 上卦：乾、坤、震、巽、坎、离、艮、兑
    "guaci": "卦辞",          // 完整的卦辞
    "yaoci": {
      "1": "初爻爻辞",        // 初爻（最下面的爻）
      "2": "二爻爻辞",        // 二爻
      "3": "三爻爻辞",        // 三爻
      "4": "四爻爻辞",        // 四爻
      "5": "五爻爻辞",        // 五爻
      "6": "上爻爻辞"         // 上爻（最上面的爻）
    }
  }
}
```

## 如何修改或补充数据

### 方法：直接编辑JSON文件

1. 打开 `core/data/hexagrams.json` 文件
2. 按照上述格式添加或修改卦数据
3. 保存文件后，重启程序即可生效（程序会在启动时自动加载）

**注意**：
- JSON文件必须使用UTF-8编码
- 确保JSON格式正确（可以使用JSON验证工具检查）
- 内卦和外卦名称必须是：乾、坤、震、巽、坎、离、艮、兑 之一

### 注意事项

1. **内卦和外卦的对应关系**：
   - 内卦（下卦）= 下三爻组成的八卦
   - 外卦（上卦）= 上三爻组成的八卦
   - 例如：水山蹇 = 坎上艮下（外卦是坎，内卦是艮）

2. **爻的顺序**：
   - 从下往上：初爻、二爻、三爻、四爻、五爻、上爻
   - 阳爻用"九"表示（初九、九二、九三...）
   - 阴爻用"六"表示（初六、六二、六三...）

3. **卦名和全名**：
   - `name`：简短的卦名（如：乾、坤、蹇）
   - `full_name`：完整的卦名（如：乾为天、水山蹇）
