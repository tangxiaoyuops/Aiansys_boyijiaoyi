# 大宗商品数据目录

本目录用于存储大宗商品分析系统的数据。

## 目录结构

```
commodity/
├── raw_evidence/      # 原始证据包（供给、价格、新闻）
├── strategies/         # 策略信号记录
└── backtest_results/   # 回测结果
```

## 目录说明

### raw_evidence/
存储从公开网站/API采集的原始数据，包括：
- 供给链数据（产能、库存、贸易流）
- 价格数据（现货、期货、指数）
- 新闻数据（政策、天气、地缘、供需）

文件命名格式：`{commodity_id}_{date}.json`

### strategies/
存储生成的买卖策略信号，包括：
- 策略类型（趋势、套利、套保、事件驱动）
- 入场点位、止损止盈
- 仓位规模、风险收益比

文件命名格式：`{strategy_id}.json`

### backtest_results/
存储策略回测结果，包括：
- 收益率、夏普比率、最大回撤
- 净值曲线、回撤曲线
- 交易记录

文件命名格式：`{strategy_id}_{backtest_date}.json`

## 数据清理

建议定期清理过期数据：
- 原始证据包：保留最近3个月
- 策略记录：保留最近6个月
- 回测结果：保留最近1年
